# Tic Tac Toe Monorepo

This monorepo contains:
- `backend/`: Bun + Hono API server for multiplayer Tic Tac Toe
- `frontend/`: SvelteKit (Svelte 5) app for the game UI
- `shared/`: Shared TypeScript types for both backend and frontend

## Getting Started

See `backend/README.md` and `frontend/README.md` for setup instructions.
